# Week 1 — JavaScript Fundamentals + CSS Basics

## What I Learned
- Variable declarations — const, let, block scope
- Primitive data types — String, Number, Boolean, Null, Undefined
- Arrays — iteration, mutation methods, index access
- Objects — key-value pairs, dot notation, bracket notation
- Functions — parameters, return values, default parameters, scope
- Array methods — forEach, map, filter, find, method chaining
- Loops — for loop, while loop
- Template literals — string interpolation
- CSS — box model, selectors, spacing, typography

## Exercises Built
- Market vendor data structures
- Price calculator with tax
- AgroChain farmer listings with filtering

## Bugs I Fixed
- .push() returns length not the array — never reassign with it
- Variables inside functions don't exist outside — block scope
- farmer vs farmers inside callbacks — parameter vs array
- Missing const/let declaration keyword — always declare variables

## Key Insight
Functions and array methods follow the same pattern —
you define what happens, JavaScript handles the repetition.
